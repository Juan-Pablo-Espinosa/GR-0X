# GR-00

Development of an opensource humanoid morphology aware reserach plataform.

## Structure
- `jetson/` — RL agent and simulation (Jetson Orin)
- `polarfire/` — ROS2 nodes for CAN bus and Robstride motor control (PolarFire SoC)
- `shared/` — Custom ROS2 msg/srv definitions shared by both boards
- `hardware/` — Arduino sketches, wiring diagrams, schematics
